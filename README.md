# 🐱 MacTata — macOS Style Desktop Chat

A Mac-style desktop interface in your browser. Chat with AI assistant Tata through this beautiful macOS-inspired UI.

- macOS-style desktop with menu bar, dock, and windows
- Real-time chat via WebSocket
- Terminal-style AI assistant
